schema do banco necessario para rodar a aplicação:

create table public.user_number (
  id bigint generated by default as identity not null,
  user_name text not null,
  phone_number bigint not null,
  country_number bigint null,
  ddd bigint null,
  active boolean not null default true,
  constraint users_numbers_pkey primary key (id)
) TABLESPACE pg_default;

como rodar o projeto:

docker build -t b2bflow .
docker run -p 80:80 b2bflow

como triggar o envio de mensagens:

curl --location 'http://0.0.0.0:8000/send_default_message'

onde os logs sao salvos:

./src/logs/exemplo.log
